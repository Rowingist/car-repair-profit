All models are set in the prefabs. You will find them in the "Prefabs" folder: "Cartoon Vehicle Pack Volume 5/Prefabs/".
Prefabs are distributed in subfolders according to the shader used. In the package you will find prefabs with free shaders “Toony Colors Free” as an example. You can read more about this shaders at the link:
https://assetstore.unity.com/packages/vfx/shaders/toony-colors-free-3926
Links to paid shader which included in the packs here:
https://assetstore.unity.com/packages/vfx/shaders/stylized-toony-shader-161170

Each of the five cars is represented by three prefabs with different shaders as an example.
All you need to do is to add the prefab to the scene.
